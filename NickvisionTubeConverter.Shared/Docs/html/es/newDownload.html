Descarga nueva

Esta página explica todas las opciones para configurar una descarga nueva en Tube Converter.

URL de los medios

La URL del medio a descargar. Debe ser un enlace http o https válido ("válido" significa sintácticamente correcto).

Si se encuentra una URL válida en el portapapeles al abrir el cuadro de diálogo Añadir descarga, la URL se pegará automáticamente en la fila de entrada de URL.

Autenticar

Si se activa, se pedirá al usuario que proporcione un nombre de usuario y/o una contraseña para pasárselos a yt-dlp al descargar.

Si el llavero está activado, se pedirá al usuario que elija de una lista de credenciales con nombre o puede elegir introducir manualmente un nombre de usuario y/o contraseña diferentes.

No todos los sitios web admiten la transmisión de nombres de usuario y contraseñas a yt-dlp. Por ejemplo, YouTube NO admite esta función.

Para estos sitios que no admiten la transmisión de credenciales, se recomienda cargar un archivo de Cookies en preferencias.

Tipo de archivo

El tipo de archivo con el que se descargarán los archivos multimedia.

Si No permitir conversiones está activada, las únicas opciones serán Vídeo (si se admite) y Audio

El tipo de archivo usado anteriormente será recordado y preseleccionado para la siguiente descarga.

Calidad

Si el tipo de archivo es de vídeo, las calidades indicadas serán las resoluciones de los medios disponibles.

Si el tipo de archivo es de audio, las calidades enumeradas serán simplemente La mejor y La peor.

La calidad usada anteriormente será recordada y preseleccionada para la siguiente descarga.

Subtítulo

El tipo de subtítulos a descargar. Puede ser Ninguno, VTT, o SRT.

Sólo disponible para tipos de vídeo.

Guardar carpeta

La carpeta en la que guardar la descarga.

Si el tipo de archivo es de audio, las calidades enumeradas serán simplemente La mejor y La peor.

La carpeta usada anteriormente será recordada y preseleccionada para la siguiente descarga.

Límite de velocidad

Si se activa, Tube Converter limitará la velocidad de descarga con la velocidad configurada en preferencias.

Disponible en la sección Opciones avanzadas.

Recortar miniatura

Sólo está disponible si Incrustar metadatos está activado en preferencias.

Si se activa, Tube Converter recortará la miniatura incrustada de una descarga como cuadrada. Esto resulta útil al descargar música cuyas imágenes de álbum suelen ser cuadradas.

Disponible en la sección Opciones avanzadas.

Descargar plazos específicos

Si se activa, el usuario podrá introducir un plazo de inicio y un plazo de finalización para que se descargue una sección de los medios.

Disponible en la sección Opciones avanzadas.

Activando esta opción se desactivará el uso de aria2 como descargador para esta descarga si está activado en preferencias.

Nombre del archivo

El nombre de archivo que se usará al guardar la descarga.

Predeterminado: El título del medio
