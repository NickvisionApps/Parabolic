New Download

This page explains all options for configuring a new download in Parabolic.

メディアのURL

The URL of the media to download. Must be a valid ("valid" meaning syntactically correct) http or https link.

If a valid URL is found inside the clipboard upon opening the Add Download dialog, the URL will be automatically pasted in the URL entry row.

認証

If enabled, the user will be asked to provide a username and/or password to pass to yt-dlp when downloading.

If keyring is enabled, the user will be asked to chose from a list of named credentials or can chose to enter manually a different username and/or password.

Not all websites support passing usernames and passwords to yt-dlp. For example, YouTube does NOT support this feature.

For these sites that don't support passing credentials, it is recommended to upload a Cookies file in Preferences instead.

ファイルの種類

The file type of which to download the media as.

If Disallow Conversions is enabled, the only options will be Video (if supported) and Audio

The previously used file type will be remembered and pre-selected for the next download.

品質

If the file type is a video type, the listed qualities will be the resolutions of the media available.

If the file type is an audio type, the listed qualities will be simply Best and Worst.

The previously used quality will be remembered and pre-selected for the next download.

字幕をダウンロードする

Whether to download and embed subtitle when downloading a video. Languages for subtitles can be defined in preferences.

The previously used state will be remembered and pre-selected for the next download.

保存フォルダー

The folder at which to save the download to.

If the file type is an audio type, the listed qualities will be simply Best and Worst.

The previously used folder will be remembered and pre-selected for the next download.

スピード制限

If enabled, Parabolic will limit the download's speed with the speed configured in Preferences.

Available in the Advanced Options section.

AV1 コーデックを優先する

Only available for video (mp4, webm) downloads

If enabled, Parabolic will prefer videos with the av1 codec when downloading.

Available in the Advanced Options section.

This option's state will be remembered and restored for future downloads.

チャプター（章）で分ける

If enabled, Parabolic will split the video into multiple smaller ones based on its chapters.

Available in the Advanced Options section.

Crop Thumbnail

Only available if Embed Metadata is enabled in Preferences.

If enabled, Parabolic will crop a download's embedded thumbnail as square. This is useful when downloading music who's album images are usually square.

Available in the Advanced Options section.

特定の時間枠をダウンロードする

If enabled, the user will be able to enter a start timeframe and an end timeframe for a section of the media to download.

Available in the Advanced Options section.

Enabling this option will disable the use of aria2 as the downloader for this download if it is enabled in Preferences.

File Name

The file name to use when saving the download.

Default: The title of the media
