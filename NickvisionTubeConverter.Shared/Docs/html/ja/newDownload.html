新規ダウンロード

このページでは Parabolic の新規ダウンロードを設定するためのすべてのオプションを解説します。

メディアの URL

メディアのダウンロード元の URL。有効な（正しい URL の構文である）http または https リンクである必要があります。

"ダウンロードを追加"のダイアログが開いた際に、クリップボードに有効な URL が見つかると、URL は自動的に URL 入力欄にペーストされます。

認証

有効であれば、ダウンロード時に、 yt-dlp に渡す、ユーザー名と（または）パスワードが要求されます。

キーリングが有効であれば、保存された資格情報の選択を要求されますが、別のユーザー名と（または）パスワードを手動で入力することもできます。

すべてのウェブサイトで yt-dlp がユーザー名とパスワードの入力に対応しているわけではありません。例えば、YouTube はこの機能にサポートしていません。

これらのサイトに資格情報は対応していない、設定でクッキーファイルをアップロードすることをお勧めします。

ファイルの種類

ダウンロードするメディアのファイルのタイプ。

フォーマット変換を無効化が有効であれば、（サポートされていれば）動画もしくは音声の二択になります

最後に使用したファイルの種類は、次回のダウンロードのために記憶・事前選択されます。

品質

ファイルの種類が動画の場合、表示される品質はメディアが利用できる解像度です。

ファイルの種類が音声の場合、表示される品質は単純に最高もしくは最低になります。

最後に使用した品質は、次回のダウンロードのために記憶・事前選択されます。

字幕をダウンロードする

動画のダウンロード時に字幕をダウンロードしてメディアに埋め込むかどうか。字幕の言語は設定にて指定できます。

最後に使用した状態は、今後のダウンロードのために記憶・事前選択されます。

保存フォルダー

ダウンロードを保存フォルダー。

ファイルの種類が音声の場合、表示される品質は単純に最高もしくは最低になります。

最後に使用したフォルダーは、次回のダウンロードのために記憶・事前選択されます。

スピード制限

有効であれば、Parabolic は、設定された速度でダウンロードの速度を制限します。

詳細オプションで選択可能です。

AV1 コーデックを優先する

動画（mp4、webm）のダウンロードのみ利用可能です

有効であれば、Parabolic は動画のダウンロード時に av1 コーデックを使用します。

詳細オプションで選択可能です。

このオプションの状態は、今後のダウンロードのために記憶・復元されます。

チャプター（章）で分ける

有効であれば、Parabolic は動画を複数の小さなチャプターごとに分割します。

詳細オプションで選択可能です。

サムネイルを整形

メタデータを埋め込むが有効な場合のみ利用可能です。

有効であれば、Parabolic はダウンロードに埋め込まれたサムネイルを正方形に整えます。アルバムイメージの音楽をダウンロードするときに便利です。

詳細オプションで選択可能です。

特定の時間枠をダウンロードする

有効であれば、ダウンロードするメディアの開始と終わりのタイムフレームを入力できるようになります。

詳細オプションで選択可能です。

このオプションを有効にすると、aria2 を使用したダウンロードが無効になります。

ファイル名

ダウンロードを保存する際に使用するファイル名。
