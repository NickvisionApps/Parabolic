Конвертер

На цій сторінці пояснюються всі параметри налаштування конвертера в Параметрах Parabolic.

Заборонити конвертацію

Якщо ввімкнено, Parabolic завантажить відповідний формат відео/аудіо для вибраної якості без конвертації в інші формати.

Це означає, що ви не зможете вибрати формат файлу під час завантаження медіафайлу. Але ви заощадите час при завантаженні, оскільки конвертер не запускатиметься.

За замовчуванням: Вимк

Вставити метадані

Якщо ввімкнено, Parabolic позначатиме медіа відповідною інформацією (назва, жанр тощо) та мініатюрою, якщо вона доступна.

Parabolic НЕ вставляє поля Коментар, Опис, Синопсис і Виворіт.

За замовчуванням: Ввімк

Обрізати мініатюри аудіо

Доступно, якщо ввімкнено Вбудовувати метадані.

Якщо ввімкнено, Parabolic автоматично вмикає додаткову параметр Обрізати мініатюру для завантаження аудіо.

За замовчуванням: Вимк

Remove Source Data

Доступно, якщо ввімкнено Вбудовувати метадані.

If enabled, Parabolic will clear metadata fields containing the URL and other identifying information of the media source.

Fields that are cleared: comment, description, synopsis, purl and handler_name in each stream.

За замовчуванням: Вимк
