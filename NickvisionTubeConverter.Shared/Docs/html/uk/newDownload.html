Нове завантаження

На цій сторінці пояснюються всі параметри налаштування нового завантаження в Tube Converter.

URL-адреса медіа

URL-адресу медіа для завантаження. Це має бути дійсне ("дійсне" означає синтаксично правильне) посилання http або https.

Якщо дійсну URL-адресу буде знайдено в буфері обміну після відкриття діалогового вікна "Додати завантаження", URL-адресу буде автоматично вставлено в рядок запису URL-адреси.

Автентифікація

Якщо ввімкнено, користувачеві буде запропоновано ввести ім'я користувача та/або пароль для входу до yt-dlp під час завантаження.

Якщо keyring ввімкнено, користувачеві буде запропоновано вибрати зі списку іменованих облікових даних або ввести вручну інше ім'я користувача та/або пароль.

Не всі вебсайти підтримують передачу імен користувачів і паролів до yt-dlp. Наприклад, YouTube НЕ підтримує цю функцію.

Для сайтів, які не підтримують передачу облікових даних, рекомендується завантажити файл Cookies в Параметрах.

Тип файлу

Тип файлу, у якому потрібно завантажити медіафайл.

Якщо ввімкнено Заборонити конвертацію, єдиними варіантами будуть Відео (якщо підтримується) та Аудіо

Раніше використаний тип файлу буде запам'ятовуватися і попередньо вибиратися для наступного завантаження.

Якість

Якщо тип файлу - відео, перелічені якості будуть відповідати роздільній здатності доступного носія.

Якщо тип файлу - аудіо, то перелічені якості будуть просто Найліпша та Найгірша.

Раніше використана якість буде запам'ятовуватися і попередньо вибиратися для наступного завантаження.

Субтитри

Тип субтитрів для завантаження. Може бути Немає, VTT, або SRT.

Доступно лише для відео.

Зберегти теку

Тека, в яку буде збережено завантаження.

Якщо тип файлу - аудіо, то перелічені якості будуть просто Найліпша та Найгірша.

Раніше використана тека буде запам'ятовуватися і попередньо вибиратися для наступного завантаження.

Обмеження швидкості

Якщо ввімкнено, Tube Converter обмежуватиме швидкість завантаження швидкістю, заданою у Параметрах.

Доступно в розділі Додаткові параметри.

Обрізати мініатюру

Доступно, якщо у Параметрах ввімкнено опцію Вставити метадані.

Якщо ввімкнено, Tube Converter обрізатиме вбудовану мініатюру завантаження до квадратної форми. Це корисно під час завантаження музики, обкладинки альбомів якої зазвичай мають квадратну форму.

Доступно в розділі Додаткові параметри.

Завантажити конкретний часовий проміжок

Якщо цю параметр ввімкнено, користувач зможе ввести початковий і кінцевий часовий проміжок для завантаження частини медіафайлу.

Доступно в розділі Додаткові параметри.

Ваімкнення цього параметру вимкне використання aria2 як завантажувача для цього завантаження, якщо його ввімкнено у Параметрах.

Ім'я файлу

Ім'я файлу, яке буде використано при збереженні завантаження.
