Nouveau téléchargement

Cette page explique toutes les options pour configurer un nouveau téléchargement dans Parabole.

URL du média

L’URL du média à télécharger. Il doit s’agir d’un lien http ou https valide (i.e.e syntaxiquement correct).

Si une URL valide est trouvée dans le presse-papiers au moment de l’ouverture de la fenêtre de dialogue Ajouter, elle sera automatiquement collée dans le champ de saisie.

S’identifier

Si cette option est activée, il sera demandé à l’utilisateur de fournir un nom d’utilisateur et/ou un mot de passe à yt-dlp lors du téléchargement.

Si le trousseau est activé, l’utilisateur devra choisir parmi une liste d’identifiants, ou entrer manuellement un nom d’utilisateur et/ou mot de passe différent.

Tous les sites web ne prennent pas en charge la fourniture de noms d’utilisateur et mots de passe à yt-dlp. Par exemple, YouTube ne prend PAS en charge cette fonctionnalité.

Pour les sites qui ne prennent pas en charge la fourniture d’identifiants, il est recommandé de téléverser un fichier de cookies dans les préférences à la place.

Type de fichier

Le type de fichier sous lequel enregistrer le média.

Si l’option Empêcher les conversions est activée, les seules options possibles seront Vidéo (si pris en charge) et Audio

Le type de fichier sélectionné précédemment sera mémorisé et présélectionné pour le prochain téléchargement.

Qualité

Si le type de fichier est une vidéo, la liste des qualités sera celle des résolutions disponibles pour le média.

Si le type de fichier est un audio, la liste des qualités sera simplement Meilleure et Basse.

La qualité utilisée précédemment sera mémorisée et présélectionnée pour le prochain téléchargement.

Télécharger les sous-titres

Indique s’il faut télécharger et inclure les sous-titres dans le fichier de la vidéo lors de son téléchargement. Les langues des sous-titres peuvent être définies dans les Préférences.

Les paramètres utilisés précédemment seront mémorisés et présélectionnés pour le prochain téléchargement.

Dossier d’enregistrement

Le dossier dans lequel enregistrer le téléchargement.

Si le type de fichier est un audio, la liste des qualités sera simplement Meilleure et Basse.

Le dossier sélectionné précédemment sera mémorisé et présélectionné pour le prochain téléchargement.

Limitation de vitesse

Si cette option est activée, Parabole limitera la vitesse de téléchargement selon la vitesse indiquée dans les préférences.

Disponible dans la section Options avancées.

Préférer le codec AV1

Seulement disponible pour les téléchargements de vidéos (mp4, webm)

Si cette option est activée, Parabole préférera les vidéos avec le codec av1 lors du téléchargement.

Disponible dans la section Options avancées.

L’état de cette option sera mémorisé et restauré pour les téléchargements futurs.

Diviser en chapitres

Si cette option est activée, Parabole séparera la vidéo en plusieurs morceaux correspondant aux chapitres qu’elles contient.

Disponible dans la section Options avancées.

Rogner la vignette

Seulement disponible si l’inclusion des métadonnées est activée dans les préférences.

Si cette option est activée, Parabole rognera la vignette d’un téléchargement en forme de carré. Ceci est utile pour le téléchargement de musiques, puisque les illustrations d’albums ont généralement cette forme.

Disponible dans la section Options avancées.

Télécharger une séquence spécifique

Si cette option est activée, l’utilisateur pourra entrer un temps de début et un temps de fin pour une partie du média à télécharger.

Disponible dans la section Options avancées.

L’activation de cette option désactivera l’utilisation de aria2 comme téléchargeur pour ce téléchargement s’il est activé dans les préférences.

Nom du fichier

Le nom du fichier à utiliser lors de l’enregistrement du téléchargement.

Par défaut : le titre du média
