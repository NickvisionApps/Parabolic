Téléchargements

Cette page explique toutes les options pour configurer les téléchargements dans les préférences de Parabolic.

Nombre maximum de téléchargements actifs

The number of downloads allowed to be running at a time. Must be an integer value between 1 and 10.

Si le maximum de téléchargements actifs est atteint, Parabolic placera les nouveaux téléchargements dans une file d’attente jusqu’à ce que des téléchargements supplémentaires soient autorisés à s’exécuter.

Par défaut : 5

Remplacer les fichiers existants

Parabolic will overwrite existing files.

Par exemple, si un fichier intitulé film.mp4 existe dans le dossier de téléchargement et qu’un nouveau téléchargement est créé avec le titre film, film.mp4 sera remplacé par le contenu du nouveau téléchargement.

Parabolic détectera les téléchargements qui devraient être repris et évitera ce paramètre lors de la récupération ou de la nouvelle tentative de téléchargement ayant échoué

Par défaut : Activée

Limiter le caractères des noms des fichiers

Parabolic will restrict characters in filenames to those supported only by Windows.

Option disponible uniquement sur Linux.

Par défaut : Désactivée

Identifiant du média inclus dans le titre après validation

Parabolic will include a media's id in its default title.

Par défaut : Activée

Inclure les sous-titres générés automatiquement

Parabolic will present auto-generated subtitles to download in addition to available subtitles.

Par défaut : Activée

Codec vidéo préféré

The video codec Parabolic should prefer when parsing media formats to download.

If Parabolic cannot find any formats with the selected preferred codec, no video formats will be available for download.

Par défaut : N’importe lequel

Preferred Audio Codec

The audio codec Parabolic should prefer when parsing media formats to download.

If Parabolic cannot find any formats with the selected preferred codec, no audio formats will be available for download.

Par défaut : N’importe lequel

Format de sous-titres préféré

The file format Parabolic should ensure all subtitles are downloaded as.

Par défaut : N’importe lequel
