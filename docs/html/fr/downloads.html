Téléchargements

Cette page explique toutes les options pour configurer les téléchargements dans les préférences de Parabolic.

Nombre maximum de téléchargements actifs

Indique le nombre de téléchargements actifs autorisés à s’exécuter en même temps. Il doit s’agir d’un nombre entier compris entre 1 et 10.

Si le maximum de téléchargements actifs est atteint, Parabolic placera les nouveaux téléchargements dans une file d’attente jusqu’à ce que des téléchargements supplémentaires soient autorisés à s’exécuter.

Par défaut : 5

Remplacer les fichiers existants

Parabolic écrasera les fichiers existants.

Par exemple, si un fichier intitulé film.mp4 existe dans le dossier de téléchargement et qu’un nouveau téléchargement est créé avec le titre film, film.mp4 sera remplacé par le contenu du nouveau téléchargement.

Parabolic détectera les téléchargements qui devraient être repris et évitera ce paramètre lors de la récupération ou de la nouvelle tentative de téléchargement ayant échoué

Par défaut : Activée

Limiter le caractères des noms des fichiers

Parabolic restreindra les caractères dans les noms de fichiers à ceux uniquement pris en charge par Windows.

Option disponible uniquement sur Linux.

Par défaut : Désactivée

Identifiant du média inclus dans le titre après validation

Parabolic inclura l'ID du media dans le titre par défaut.

Par défaut : Activée

Inclure les sous-titres générés automatiquement

Parabolic présentera des sous-titres auto-générés a télécharger en plus des sous-titres disponibles.

Par défaut : Activée

Codec vidéo préféré

Le codec vidéo que Parabolic devrait préférer lors de la sélection de format de médias à télécharger.

Si Parabolic ne peut pas trouver un format avec le codec préféré, aucun format vidéo ne sera disponible a télécharger.

Par défaut : N’importe lequel

Codec audio préféré

Le codec audio que Parabolic devrait préférer lors de la sélection de format de médias à télécharger.

Si Parabolic ne peut pas trouver un format avec le codec préféré, aucun format audio ne sera disponible a télécharger.

Par défaut : N’importe lequel

Format de sous-titres préféré

Le format de fichier que Parabolic devrais assurer que tous les sous-titres téléchargés son.

Par défaut : N’importe lequel
