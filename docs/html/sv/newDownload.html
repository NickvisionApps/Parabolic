Ny nedladdning

Den här sidan förklarar alla alternativ för att konfigurera en ny nedladdning i Parabolic.

Media URL

URL:en till mediet som ska laddas ner. Måste vara en giltig (”giltig” vilket betyder syntaktisk korrekt) http- eller https-länk.

Om en giltig URL hittas i urklippet när dialogrutan "Lägg till nedladdning" öppnas, kommer URL:en automatiskt att klistras in i URL-inmatningsraden.

Autentisera

Om det är aktiverat kommer användaren att bli ombedd att ange ett användarnamn och/eller lösenord för att skicka till yt-dlp vid nedladdning.

If credentials are found in the keyring, the user can instead chose from a list of named credentials to use (or continue to manually enter a credential)

Alla webbplatser stöder inte att skicka användarnamn och lösenord till yt-dlp. Till exempel stöder YouTube INTE den här funktionen.

For these sites that don't support passing credentials, it is recommended to select a browser to use cookies from in Preferences instead.

Filtyp

Den filtyp som mediet ska laddas ner som.

Om Tillåt inte konverteringar är aktiverat, är de enda alternativen Video (om stöds) och Ljud

Den tidigare använda filtypen kommer att kommas ihåg och blir förvald för nästa nedladdning.

Kvalitet

Om filtypen är en videotyp kommer de listade kvaliteterna att vara upplösningarna för det tillgängliga mediet.

If the file type is an audio type, the listed qualities will be simply Best, Good, and Worst.

Den tidigare använda kvaliteten kommer att kommas ihåg och blir förvald för nästa nedladdning.

Audio Language

The language of which to use from the audio part of a media download.

If no additional languages are available, Default will be the only option.

Det tidigare använda tillståndet kommer att kommas ihåg och blir förvalt för nästa nedladdning.

Ladda ner undertext

Whether to download and embed subtitle when downloading a video. Further subtitle settings can be configured in preferences.

Det tidigare använda tillståndet kommer att kommas ihåg och blir förvalt för nästa nedladdning.

Föredrar AV1 Codec

Om det är aktiverat kommer Parabolic att föredra videor med AV1-codec vid nedladdning.

Det tidigare använda tillståndet kommer att kommas ihåg och blir förvalt för nästa nedladdning.

Dela upp kapitlen

If enabled, Parabolic will split a video download into multiple smaller files based on its chapters.

Det tidigare använda tillståndet kommer att kommas ihåg och blir förvalt för nästa nedladdning.

Lagringsmapp

Mappen där nedladdningen ska sparas.

Den tidigare använda mappen kommer att kommas ihåg och blir förvald för nästa nedladdning.

Filnamn

Filnamnet som ska användas när nedladdningen sparas.

Standard: Mediets titel

Time Frame

The user can enter a start time frame and an end time frame to download a specific section of the media.

Default: The whole media length

Enabling this option will disable the use of aria2 as the downloader for this download, even if it is enabled in Preferences.

Hastighetsgräns

If enabled, Parabolic will limit the download's speed with the speed configured in Preferences.

Tillgängligt i avsnittet Avancerade alternativ.
