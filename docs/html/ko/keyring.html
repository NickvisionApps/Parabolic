비밀번호 저장소

이 페이지는 Parabolic에서 키링을 사용하는 모든 옵션에 대해 설명합니다.

백엔드

키링 기능은 libnick 라이브러리에 의해 지원됩니다.

이 라이브러리는 SQLCipher를 사용하여 자격 증명을 플랫폼에 독립적으로 안전하게 저장합니다.

이 라이브러리는 시스템의 자격 증명 관리자를 사용하여 키링 데이터베이스 파일을 보호합니다. 예를 들어, Linux에서는 DBus 비밀 서비스, Windows에서는 Windows 자격 증명 관리자를 사용합니다.

자격 증명 저장

자격 증명에 저장할 수 있는 필드는 다음과 같습니다:

Name: 자격 증명의 이름

URL (선택 사항): 자격 증명의 URL

Username (비밀번호가 비어 있지 않으면 선택 사항): 자격 증명의 사용자 이름

Password (사용자 이름이 비어 있지 않으면 선택 사항): 자격 증명의 비밀번호
