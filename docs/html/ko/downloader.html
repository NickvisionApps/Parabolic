다운로더

이 페이지는 Parabolic의 설정에서 다운로더를 구성하는 모든 옵션에 대해 설명합니다.

Use Part Files

Parabolic will download media in separate .part files instead of directly into the output file.

기본값: 켬

YouTube에서 SponsorBlock 사용

Parabolic will attempt to remove sponsored segments from YouTube videos.

SponsorBlock은 YouTube 비디오에서만 작동합니다. 그러나 이 설정을 활성화해도 다른 사이트에서의 다운로드에는 영향을 미치지 않으며(즉, 실패하지 않음) 안전하게 사용할 수 있습니다.

기본값: 꺼짐

속도 제한

The limit, in KiB/s, to apply to downloads only with the advanced option Speed Limit enabled. Must be an integer value between 512 and 10240.

기본값: 1024

프록시 URL

사용할 프록시 서버의 URL입니다.

기본값: URL 없음

브라우저의 쿠키

yt-dlp의 다운로드 엔진에 전달할 쿠키를 읽을 브라우저입니다.

이 설정은 로그인 요구 미디어(예: 개인 YouTube 비디오)를 다운로드할 때 유용할 수 있습니다.

이 옵션은 샌드박스되지 않은 플랫폼에서만 사용할 수 있습니다.

기본값: 없음

쿠키 파일

다운로드 엔진에서 yt-dlp에 전달할 쿠키의 .txt 파일입니다.

이 설정은 로그인 요구 미디어(예: 개인 YouTube 비디오)를 다운로드할 때 유용할 수 있습니다.

브라우저에서 쿠키 파일을 얻기 위해 다음 확장 프로그램 중 하나를 사용하세요:

Chrome: https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc

Firefox: https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/

이 확장 프로그램은 사용자의 책임 하에 사용하세요.

기본값: 선택된 파일 없음
