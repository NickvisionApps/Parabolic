다운로드

이 페이지는 Parabolic의 설정에서 다운로드를 구성하는 모든 옵션에 대해 설명합니다.

활성 다운로드 최대 수

The number of downloads allowed to be running at a time. Must be an integer value between 1 and 10.

활성 다운로드의 최대 수에 도달하면, Parabolic은 새 다운로드를 대기열에 배치하고 더 많은 다운로드가 실행될 수 있을 때까지 기다립니다.

기본값: 5

기존 파일 덮어쓰기

Parabolic will overwrite existing files.

예를 들어, 다운로드 폴더에 movie.mp4라는 파일이 이미 존재하고, 새 다운로드가 movie라는 이름으로 생성되면, movie.mp4 파일은 새 다운로드의 내용으로 덮어씌워집니다.

Parabolic은 재개해야 할 다운로드를 감지하고 실패한 다운로드를 복구하거나 재시도할 때 이 설정을 피합니다

기본값: 켬

파일명 글자 수 제한

Parabolic will restrict characters in filenames to those supported only by Windows.

이 옵션은 Linux에서만 사용할 수 있습니다.

기본값: 꺼짐

검증 후 제목에 미디어 ID 포함

Parabolic will include a media's id in its default title.

기본값: 켬

자동 생성된 자막 포함

Parabolic will present auto-generated subtitles to download in addition to available subtitles.

기본값: 켬

선호하는 비디오 코덱

The video codec Parabolic should prefer when parsing media formats to download.

If Parabolic cannot find any formats with the selected preferred codec, no video formats will be available for download.

기본값: 아무거나

Preferred Audio Codec

The audio codec Parabolic should prefer when parsing media formats to download.

If Parabolic cannot find any formats with the selected preferred codec, no audio formats will be available for download.

기본값: 아무거나

선호하는 자막 형식

The file format Parabolic should ensure all subtitles are downloaded as.

기본값: 아무거나
