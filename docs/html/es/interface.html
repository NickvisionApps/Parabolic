Interfaz de usuario

Esta página explica todas las opciones para configurar la interfaz de usuario en Preferencias en Parabólico .

Tema

El tema para la interfaz de usuario.

Por defecto: el del sistema

Disparador de las notificaciones de completado

El mecanismo utilizado para mostrar notificaciones de finalización de descarga cuando la aplicación no está activa.

Predeterminado: para cada descarga

Prevenir suspensión

Si está habilitado, se evitará que el sistema se suspenda mientras Parabolic esté en ejecución.

Predeterminado: Apagado

Recuperar descargas fallidas

Si está habilitado, Parabolic rastreará las descargas en ejecución y en cola y las reiniciará si es necesario, si la aplicación se bloquea y se reinicia.

Predeterminado: Encendido

Descargar inmediatamente después de la validación

Evita la selección de opciones de descarga en el cuadro de diálogo Agregar descarga e inmediatamente descarga un medio validado utilizando las opciones previamente seleccionadas.

Predeterminado: Apagado

Tamaño del historial de descargas

El período de tiempo para conservar el historial de descargas.

Predeterminado: una semana
