Benutzeroberfläche

This page explains all options for configuring the User Interface in Preferences in Parabolic.

Thema

Das Thema für die Benutzeroberfläche.

Standardeinstellung: System

Completed Notification Trigger

The trigger used for showing completed download notifications when the application is not active.

Standardeinstellung: Für jeden Download

Unterbrechung verhindern

Wenn aktiviert, wird verhindert, dass das System während der Ausführung von Parabolic in den Ruhezustand versetzt wird.

Standardeinstellung: Aus

Recover Crashed Downloads

If enabled, Parabolic will track the running and queued downloads and restart them if necessary if the application crashes and is restarted.

Standardeinstellung: Ein

Unmittelbar nach der Validierung herunterladen

Bypasses the download options selection in the Add Download dialog and immediately downloads a validated media using previously selected options.

Standardeinstellung: Aus

Länge des Herunterladeverlaufs

Die Zeitspanne, in der der Herunterladeverlauf gespeichert wird.

Standardeinstellung: Eine Woche
