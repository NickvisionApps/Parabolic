Downloads

This page explains all options for configuring Downloads in Preferences in Parabolic.

Høyeste antall aktive nedlastinger

Det høyeste antallet nedlastinger som kan gjennomføres samtidig. Må være et tall mellom 1 og 10.

Hvis det høyeste antallet aktive nedlastinger nås, vil Parabolic sette resten i en kø til flere nedlastinger kan gjennomføres.

Standard: 5

Erstatt eksisterende filer

Hvis dette er på, vil Parabolic kunne erstatte eksisterende filer ved navnekollisjoner.

Hvis en fil som heter video.mp4 ligger i nedlastingsmappen, og en ny nedlasting ved navn video blir lagt til, vil video.mp4 erstattes med den nye filen.

Parabolic will detect downloads that should be resumed and avoid this setting when recovering or retrying failed downloads

Standard: på

Limit Filename Characters

Restricts characters in filenames to only those supported by Windows.

This option is only available on Linux.

Standard: av

Included Media Id in Title After Validation

Whether or not to included the media's id in the title (file name) when validated.

Standard: på

Inkluder automatiske undertekster

Om automatisk genererte undertekster som samsvarer med det valgte språket skal inkluderes i nedlastingen.

Standard: på

Preferred Video Codec

The video codec Parabolic should prefer when selecting media formats to download.

Default: Any

Preferred Subtitle Format

The file format Parabolic should ensure all downloaded subtitles are downloaded as.

Default: Any
