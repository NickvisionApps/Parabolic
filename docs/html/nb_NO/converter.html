Konvertering

Denne seksjonen tar for seg alle innstillingene for konvertering i Parabolic.

Bygg inn metadata

If enabled, Parabolic will tag a media with its relevant information (title, genre, etc...) if available.

Standard: på

Fjern kildedata

Hvis dette er på, vil Parabolic fjerne alle metadata om kilden til videoen, for eksempel nettadresser.

Felter som fjernes:comment, description, synopsis, purl og handler_name på hver nedlasting.

Denne innstillingen kan bare endres hvis «Bygg inn metadata» er på.

Standard: av

Embed Thumbnails

If enabled, Parabolic will embed a media with its thumbnail if available.

If the file type does not support embedding, the thumbnail will be written to a separate image file.

Standard: på

Kvadratiske miniatyrbilder

Hvis dette er på, vil Parabolic beskjære miniatyrbilder slik at de blir kvadratiske.

This option is only configurable if Embed Thumbnails is on.

Standard: av

Bygg inn kapitler

Hvis dette er på, vil Parabolic forsøke å legge ved informasjon om kapitler i videoen.

Standard: av

Bygg inn undertekster

If disabled or if embedding is not supported, downloaded subtitles will be saved to a separate file.

Standard: på

Behandlingstråder

Antall tråder som kan brukes til etterbehandling av hver nedlasting (med FFmpeg).

Standard: Det maksimale antallet tråder som er tilgjengelig på systemet
