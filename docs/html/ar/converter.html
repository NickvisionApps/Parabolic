محوّل الصيغة

توضح هذه الصفحة جميع الخيارات لتكوين المحول في التفضيلات في Parabolic.

دمج بيانات الوصف

If enabled, Parabolic will tag a media with its relevant information (title, genre, etc...) if available.

الإفتراضي: مفعل

أزل بيانات المصدر

عند التفعيل، بارابولِك سيتم مسح حقول البيانات الوصفية التي تحتوي على عنوان URL والمعلومات التعريفية الأخرى للوسائط.

الحقول اللي تم مسحها: التعليق، الوصف، الملخص، الرابط، واسم المعالج في كل تدفق.

هذا الخيار قابل للتعديل عندما يكون Embed Metadataمفعل.

الافتراضي: إيقاف

دمج صورة العرض

If enabled, Parabolic will embed a media with its thumbnail if available.

إن لم يدعم نوع الملف هذا الدمج، صورة العرض ستُكتب كصورة مُستقلّة.

الإفتراضي: مفعل

قص صور الصوتيات المصغرة

عند التفعيل، بارابولِك سيقوم بقص الصور المصغرة لملفات الصوت لجعلها ذات ابعاد مربعة.

This option is only configurable if Embed Thumbnails is on.

الافتراضي: إيقاف

دمج الفصول

عند التفعيل، بارابولِك سيقوم بوضع علامة للمحتوى بمعلومات القسم ان وجدت.

الافتراضي: إيقاف

دمج الترجمة

إن أُطفِئ الدمج أو لم يُدعم، الترجمة المُحمّلة ستُحفظ كملف مستقل.

الإفتراضي: مفعل

مسارات مابعد المعالجة

عدد مؤشرات الترابط (threads) المسموح للمعالج اللاحق (FFmpeg) باستخدامها لكل عملية تنزيل.

الافتراضي: الحد الأقصى لعدد مؤشرات الترابط المتاحة في النظام
