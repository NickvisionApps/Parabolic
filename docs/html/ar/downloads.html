التنزيلات

توضح هذه الصفحة جميع الخيارات لإعداد التنزيلات في التفضيلات داخل تطبيق Parabolic.

الحد الأقصى لعدد التنزيلات النشطة

The number of downloads allowed to be running at a time. Must be an integer value between 1 and 10.

إذا وصل عدد التحميلات النشطة للحد الأقصى، بيحط تطبيق Parabolic التحميلات الجديدة في قائمة الانتظار لين يصير فيه مجال لتحميلات جديدة تشتغل.

الأفتراضي: 5

أُكتُب فوق الملفات الموجودة

Parabolic will overwrite existing files.

For example, if a file named movie.mp4 exists in the download folder, and a new download is created named movie, movie.mp4 will be overwritten with the new download's content.

تطبيق Parabolic بيكشف التحميلات اللي لازم تُستأنف، وبيتجاوز هالإعداد إذا كان يسترجع أو يحاول يعيد تحميلات فشلت

الإفتراضي: مفعل

تحديد عدد أحرف اسم الملف

Parabolic will restrict characters in filenames to those supported only by Windows.

هذا الخيار متوفر فقط لنسخة لينكس.

الافتراضي: إيقاف

معرف الوسائط مضمن في العنوان بعد التحقق

(بارا-بوليك) سيُرفق معرّف الملف المرئي في تسميتها الإفتراضية.

الإفتراضي: مفعل

يتضمن ترجمة تم انشائها تلقائياً

(بارا-بوليك) سيُظهر الترجمة التلقائية للتحميل، فوق الترجمات المتاحة.

الإفتراضي: مفعل

تفضيل ترميز المقطع

ترميز المقطع الذي يُفضِّلُه (بارا-بوليك) أثناء نحليل صيغ المرئيات للتحميل.

إن لم يجد بارابوليك أي صيغة بالترميز المحدد الذي إخترتَه، لن يكون هناك صيغة مقطع متوفرة للتنزيل.

القيمة الافتراضية: أي شيء

ترميز الصوت المفضل

ترميز الصوت الذي يُفضِّلُه (بارا-بوليك) أثناء نحليل صيغ المرئيات للتحميل.

If Parabolic cannot find any formats with the selected preferred codec, no audio formats will be available for download.

القيمة الافتراضية: أي شيء

Preferred Subtitle Format

صيغة الملف الذي يضمنه (بارا-بوليك) بتحميل كل الترجمات.

القيمة الافتراضية: أي شيء
