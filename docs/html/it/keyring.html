Portachiavi

Questa pagina illustra tutte le opzioni per usare il Portachiavi in Parabolic.

Backend

La funzionalità Portachiavi è supportata dalla libreria libnick.

Questa libreria usa SQLCipher per memorizzare in modo sicuro le credenziali, in maniera indipendente dalla piattaforma.

La libreria userà il gestore di credenziali del sistema (come DBus in Linux o il Gestore delle credenziali inu Windows) per proteggere il file del database del portachiavi.

Memorizza una credenziale

Ecco i campi che possono essere memorizzati in una credenziale:

Nome: il nome di una credenziale

URL (opzionale): l'URL di una credenziale

Nome utente (opzionale se il campo della password non è vuoto): il nome utente di una credenziale

Password (opzionale se il campo del nome utente non è vuoto): la password di una credenziale
