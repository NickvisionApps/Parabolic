Nuovo download

Questa pagina illustra tutte le opzioni per configurare un nuovo download in Parabolic.

URL media

L'URL del contenuto da scaricare. Deve essere un collegamento http o https valido (cioè sintatticamente corretto).

Se, all’apertura della finestra 'Aggiungi download', viene rilevato un URL valido negli appunti, questo verrà incollato automaticamente nel campo URL.

Usa file batch

Seleziona un file batch di testo contenente un elenco di URL da convalidare.

Ogni URL può essere accompagnata da una cartella di salvataggio opzionale (separata da |) in cui salvare i contenuti multimediali corrispondenti.

Tutti gli URL e i relativi contenuti multimediali presenti nel file batch verranno raggruppati e trattati da Parabolic come un'unica playlist.

La cartella di salvataggio opzionale deve essere un percorso assoluto.

Formato file batch:

URL1

URL2 | "Cartella1"

"URL3"|"Cartella2"

"URL4"

Autenticati

Se questa opzione è abilitata, verrà chiesto all'utente di inserire un nome utente e/o una password da passare a yt-dlp durante i download.

Se nel portachiavi sono presenti delle credenziali, l’utente può invece scegliere da un elenco di credenziali da usare (o continuare ad inserirle manualmente)

Non tutti i siti supportano il passaggio di nome utente e password a yt-dlp. Per esempio, YouTube NON supporta questa funzione.

Per i siti che non supportano il passaggio delle credenziali, ti consigliamo invece di selezionare nelle Preferenze un browser da cui usare i cookie.

Download immediato

Parabolic scaricherà immediatamente il media dopo la validazione, usando le opzioni per il download configurate precedentemente.

Predefinito: disattivato

Tipo dl file

Il tipo del file con cui scaricare il media.

Il tipo di file precedentemente usato verrà memorizzato e pre-selezionato per il prossimo download.

Selezionando i tipi di file Video (generico) o Audio (generico) Parabolic scaricherà il media nel formato originale senza eseguire conversioni.

I tipi di file generici non supportano pienamente le miniature e i sottotitoli. Se l'incorporamento non è supportato verranno salvati file separati.

Formato video

L'elenco dei formati video disponibili per il download, ordinato per risoluzione e bitrate.

Ogni formato è nella forma: risoluzione | bitrate | codec (ID).

Se il tipo di file è audio, questa opzione non sarà disponibile.

Il formato video precedentemente usato sarà memorizzato e pre-selezionato per il prossimo download.

I seguenti sono formati speciali usati da Parabolic per abbinare i comportamenti di yt-dlp:

- Nessuno = nessuna opzione passata a yt-dlp

- Migliore = bv*

- Peggiore = wv*

Formato audio

L'elenco dei formati audio disponibili per il download, ordinato per bitrate.

Ogni formato è nella forma: bitrate | Lingua (ID).

Il formato audio precedentemente usato sarà memorizzato e pre-selezionato per il prossimo download.

I seguenti sono formati speciali usati da Parabolic per abbinare i comportamenti di yt-dlp:

- Nessuno = nessuna opzione passata a yt-dlp

- Migliore = ba

- Peggiore = wa

Sottotitoli

L'elenco dei sottotitoli disponibili dai media che possono essere selezionati per il download dall'utente. Ulteriori impostazioni di sottotitoli possono essere configurate in Preferenze .

L'opzione sottotitoli precedentemente selezionata sarà memorizzata e pre-selezionata per il prossimo download.

Dividi capitoli

Se questa opzione è abilitata, Parabolic dividerà un download di un video in più file più piccoli in base ai suoi capitoli.

Lo stato precedentemente usato sarà memorizzato e pre-selezionato per il prossimo download.

Esporta descrizione

Se questa opzione è abilitata, Parabolic esporterà la descrizione del media in un file .description insieme al file scaricato.

Lo stato precedentemente usato sarà memorizzato e pre-selezionato per il prossimo download.

Escludi dalla cronologia

Se questa opzione è abilitata, Parabolic escluderà questo download dall'elenco cronologia download.

Argomento post eleaborazione

Un argomento da passare al post processore durante il download. Gli argomenti di post elaborazione disponibili possono essere configurati in Preferenze.

L'argomento precedentemente selezionato sarà memorizzato e pre-selezionato per il prossimo download.

Scrivi file playlist M3U

Se questa opzione è abilitata, Parabolic salverà un file M3U per il download della playlist.

Lo stato precedentemente usato sarà memorizzato e pre-selezionato per il prossimo download di playlist.

Intervallo di tempo

L'utente può inserire un periodo iniziale e finale per il download di una sezione specifica del media.

Predefinito: l'intera lunghezza dei media

Cartella salvataggio

La cartella in cui salvare il download.

La cartella precedentemente usata verrà memorizzata e pre-selezionata per il prossimo download.

Nome file

Il nome del file da usare quando si salva il download.

Predefinito: il titolo dei media
