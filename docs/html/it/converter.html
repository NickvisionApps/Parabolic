Convertitore

Questa pagina spiega tutte le opzioni per configurare il convertitore nelle Preferenze in Parabolic.

Incorpora metadati

Parabolic taggherà un media con le sue informazioni rilevanti (titolo, genere, ecc...) se disponibili.

Predefinito: ON

Rimuovi dati sorgente

Parabolic eliminerà i metadatio con le informazioni identificative.

Informazioni eliminate in ogni stream: commento, descrizione, sinossi, rpurl e handler_name.

L'opzione è configurabile solo se è abilitata l'opzione Incorpora metadati.

Predefinito: disattivato

Incorpora miniature

Parabolic incorporerà un media con la relativa miniatura, se disponibile.

Se il tipo di file non supporta l'incorporamento, la miniatura verrà salvata in un file immagine separato.

Predefinito: ON

Ritaglia miniature audio

Parabolic ritaglierà le copertine degli audio in dei quadrati.

Questa opzione è configurabile solamente se è attiva l'opzione Incorpora miniatura.

Predefinito: disattivato

Incorpora capitoli

Parabolic etichetterà un media con le sue informazioni sui capitoli, se disponibili.

Predefinito: disattivato

Incorpora sottotitoli

Se disabilitati o se l'incorporamento non è supportato, i sottotitoli scaricati verranno salvati in file separati.

La possibilità di incorporare i sottotitoli dipende dal Formato sottotitoli preferito selezionato.

Predefinito: ON

Thread FFmpeg

Parabolic limiterà il numero di thread usati da FFmpeg.

Predefinito: numero massimo di thread disponibili nel sistema

Argomenti post elaborazione

Parabolic visualizzerà queste opzioni nella finestra di dialogo Aggiungi download da passare a FFmpeg e altri post elaboratori durante l'elaborazione dei media.

Campi argomento post elaborazione:

Nome - Il nome amichevole dell'argomento.

Post processore - Il programma di post elaborazione a cui trasmettere gli argomenti.

Eseguibile - L'eseguibile a cui passare gli argomenti.

Arg - Gli argomenti da passare.
