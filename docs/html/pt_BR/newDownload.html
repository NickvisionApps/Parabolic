Novo Download

Esta página explica todas as opções para configurar um novo download no Parabolic.

URL da mídia

A URL da mídia a ser baixada. Deve ser um link http ou https válido ("válido" significa sintaticamente correto).

Se uma URL válida for encontrada na área de transferência ao abrir a caixa de diálogo Adicionar download, a URL será colada automaticamente na linha de entrada de URL.

Usar arquivo em lote

Selecione um arquivo de texto em lote que contenha uma lista de URLs para validar.

Cada URL pode ser acompanhada por uma pasta de salvamento opcional (delimitada por |) na qual salvar cada mídia da URL.

Todos os URLs e suas mídias do arquivo em lote serão agrupados e tratados como uma única lista de reprodução pelo Parabolic.

A pasta de salvamento opcional deve ser um caminho absoluto.

Formato de arquivo em lote:

URL1

URL2 | "Pasta1"

"URL3"|"Pasta2"

"URL4"

Autenticar

Se ativado, o usuário será solicitado a fornecer um nome de usuário e/ou senha para passar ao yt-dlp durante o download.

Se forem encontradas credenciais no chaveiro, o usuário pode escolher entre uma lista de credenciais nomeadas para usar (ou continuar a inserir manualmente uma credencial)

Nem todos os sites oferecem suporte à transmissão de nomes de usuário e senhas para o yt-dlp. Por exemplo, o YouTube NÃO oferece suporte a esse recurso.

Para esses sites que não suportam a passagem de credenciais, é recomendável selecionar um navegador para usar cookies em Preferências.

Baixar imediatamente

O Parabolic baixará a mídia imediatamente após a validação, usando as opções de download configuradas anteriormente.

Padrão: Desligado

Tipo de arquivo

O tipo de arquivo do qual a mídia será baixada.

O tipo de arquivo usado anteriormente será lembrado e pré-selecionado para o próximo download.

Selecionar os tipos de arquivo Vídeo (Genérico) ou Áudio (Genérico) fará com que o Parabolic baixe a mídia no tipo de arquivo original sem realizar nenhuma conversão.

Tipos de arquivo genéricos não oferecem suporte total à incorporação de miniaturas e legendas. Arquivos separados podem ser gravados em disco se a incorporação não for suportada.

Formato de vídeo

A lista de formatos de vídeo disponíveis para download, classificados por resolução e taxa de bits.

Each format is the form: resolution | bitrate | codec (id).

If the file type is an audio type, this option will not be available.

O formato de vídeo usado anteriormente será relembrado e pré-selecionado no próximo download.

The following are special formats used by Parabolic to match yt-dlp behaviors:

- None = No option passed to yt-dlp

- Best = bv*

- Worst = wv*

Formato de áudio

The list of audio formats available for download, sorted by bitrate.

Each format is the form: bitrate | language (id).

O formato de áudio usado anteriormente será relembrado e pré-selecionado no próximo download.

The following are special formats used by Parabolic to match yt-dlp behaviors:

- None = No option passed to yt-dlp

- Best = ba

- Worst = wa

Legendas

The list of subtitles available by the media that can be selected for download by the user. Further subtitle settings can be configured in preferences.

O formato de legendas usado anteriormente será relembrado e pré-selecionado no próximo download.

Dividir capítulos

If enabled, Parabolic will split a video download into multiple smaller files based on its chapters.

The previously used state will be remembered and pre-selected for the next download.

Exportar descrição

Se ativado, o Parabolic vai exportar a descrição da mídia em um arquivo .description ao lado do arquivo baixado.

The previously used state will be remembered and pre-selected for the next download.

Apagar do histórico

Se ativado, o Parabolic excluirá o download da lista de histórico de downloads.

Argumento do pós-processador

An argument to pass to the post processor during the download. The available postprocessing arguments can be configured in preferences.

The previously selected argument will be remembered and pre-selected for the next download.

Gravar arquivo de playlist M3U

If enabled, Parabolic will write a m3u file for the playlist download.

The previously used state will be remembered and pre-selected for the next playlist download.

Time Frame

The user can enter a start time frame and an end time frame to download a specific section of the media.

Default: The whole media length

Pasta de destino

The folder at which to save the download to.

The previously used folder will be remembered and pre-selected for the next download.

Nome do arquivo

The file name to use when saving the download.

Default: The title of the media
