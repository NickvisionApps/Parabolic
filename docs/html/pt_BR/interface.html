Interface do usuário

Esta página explica todas as opções para configurar a Interface de Usuário nas Preferências no Parabolic.

Tema

O tema para a interface de usuário.

Padrão: Sistema

A ação para notificação de conclusão

A ação que ativa notificações de downloads concluídos quando o programa não estiver ativo.

Padrão: Para cada download

Prevenir suspender

Se ativado. o sistema será impedido de suspender enquanto o Parabolic estiver em execução.

Padrão: Desligado

Recuperar downloads com falha

Se ativado, o Parabolic rastreará os downloads em execução e na fila e os reiniciará, se necessário, caso o programa trave e seja reiniciado.

Padrão: Ativado

Baixar imediatamente após a validação

Bypasses the download options selection in the Add Download dialog and immediately downloads a validated media using previously selected options.

Padrão: Desligado

Duração do histórico de downloads

The amount of time to keep download history saved.

Padrão: Uma semana
