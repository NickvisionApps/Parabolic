金鑰環

此頁面解釋使用 Parabolic 金鑰圈的選項。

後端

金鑰圈的功能由 libnick 函式庫提供。

此函式庫使用 SQLCipher 來安全的儲存身份驗證資訊，不論在哪個平臺上都能用。

函式庫會使用系統的身份驗證管理員 (像是 Linux 的 DBus secret service 和 Windows 的認證管理員) 以確保金鑰圈資料庫安全。

儲存身份驗證資訊

這些是可以儲存在身份驗證資訊的欄位：

名稱：身份驗證資訊的名稱

URL (選擇性)：身份驗證資訊的 URL

使用者名稱 (如果密碼不是空白則爲選填)：身份驗證資訊的使用者名稱

密碼 (如果使用者名稱不是空白則爲選填)：身份驗證資訊的密碼
