Konwersja

Na tej stronie wyjaśniono wszystkie opcje dostępne w sekcji Konwersja w preferencjach w Parabolic.

Nie zezwalaj na konwersję

Po włączeniu, Parabolic pobierze wideo i audio w formacie odpowiednim dla wybranej jakości bez przeprowadzania konwersji na inne formaty.

To oznacza, że nie będzie można wybrać formatu pliku podczas pobierania, ale konwesja nie będzie musiała zostać przeprowadzona co pozwoli zaoszczędzić czas.

Domyślnie: wyłączone

Zapisuj metadane w plikach

Po włączeniu, Parabolic będzie oznaczać multimedia odpowiednimi informacjami (tytułem, gatunkiem, itd.) i miniaturą, jeśli jest dostępna.

Parabolic NIE zapisze w pliku komentarzy, opisu, podsumowania czy purl.

Domyślnie: włączone

Przycinaj miniatury plików audio

Dostępne wyłącznie, jeśli opcja "Zapisuj metadane w plikach" jest włączona.

Po włączeniu, Parabolic automatycznie włączy opcję zaawansowaną Przytnij miniaturę dla plików audio.

Domyślnie: wyłączone

Usuń dane źródłowe

Dostępne wyłącznie, jeśli opcja "Zapisuj metadane w plikach" jest włączona.

If enabled, Parabolic will clear metadata fields containing the URL and other identifying information of the media source.

Fields that are cleared: comment, description, synopsis, purl and handler_name in each stream.

Domyślnie: wyłączone

Zapisuj rozdziały w plikach

If enabled, Parabolic will tag a media with chapter information if available.

Domyślnie: wyłączone

Embed Subtitles

If disabled or if embedding fails, downloaded subtitles will be saved to a separate file.

Domyślnie: włączone
