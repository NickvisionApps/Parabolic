Конвертер

На данной странице описаны все параметры по конфигурации конвертера в настройках Parabolic.

Встраивание метаданных

Parabolic по возможности будет добавлять в медиа теги с основными данными (заголовок, жанр и т.д.).

По умолчанию: Вкл

Удаление данных об источнике

Parabolic будет стирать поля метаданных, содержащие идентифицирующие сведения о загрузке.

В каждой дорожке будут стёрты следующие поля: comment, description, synopsis, purl и handler_name.

Данный параметр доступен, только если включено Встраивание метаданных.

По умолчанию: Выкл

Встраивание обложек

Parabolic по возможности будет встраивать обложку медиа в файл.

Если тип файла не поддерживает встраивание, обложка сохранится в виде отдельного изображения.

По умолчанию: Вкл

Обрезка обложки аудио

Parabolic будет обрезать обложки аудиофайлов до квадратного разрешения.

Данный параметр доступен, только если включено Встраивание обложек.

По умолчанию: Выкл

Встраивание глав

Parabolic по возможности будет добавлять в медиа теги с главами.

По умолчанию: Выкл

Встраивание субтитров

Если параметр выключен или если встраивание не поддерживается, загруженные субтитры сохранятся в отдельных файлах.

Поддержка встраивания зависит от выбранного вами параметра Предпочтительный формат субтитров.

По умолчанию: Вкл

Потоки FFmpeg

Parabolic будет ограничивать количество потоков, выделяемых ffmpeg.

По умолчанию: Максимальное количество потоков, доступных в системе

Аргументы постобработки

Parabolic предоставит следующие параметры в диалоге добавления загрузки для последующей передачи ffmpeg и другим постпроцессорам при обработке медиа.

Поля аргументов постобработки:

Имя — Описательное имя аргумента.

Постпроцессор — Постпроцессор, которому передаются аргументы.

Исполняемый файл — Внешняя программа, которой передаются аругменты.

Переменные — Непосредственно передаваемые переменные.
