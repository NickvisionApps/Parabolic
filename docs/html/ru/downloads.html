Загрузки

На данной странице описаны все параметры по конфигурации загрузок в настройках Parabolic.

Максимум активных загрузок

Допустимое количество одновременно выполняемых загрузок. Должно быть целое число от 1 до 10.

При достигнутом максимуме активных загрузок Parabolic будет ставить новые загрузки в очередь до тех пор, пока не освободится возможность для их запуска.

По умолчанию: 5

Перезапись имеющихся файлов

Parabolic будет перезаписывать уже существующие файлы.

Например, если файл с именем фильм.mp4 существует в папке сохранения, а новый файл создаётся с именем фильм, файл фильм.mp4 будет перезаписан новым загружаемым файлом.

Parabolic будет отслеживать загрузки, которые следует возобновить, и избегать данный параметр при восстановлении или повторном запуске неудачных загрузок

По умолчанию: Вкл

Ограничение символов в имени файла

Parabolic будет ограничивать имена файлов только теми символами, которые поддерживаются Windows.

Данный параметр доступен только в Linux.

По умолчанию: Выкл

Добавление ID медиа в заголовок

Parabolic будет добавлять идентификатор медиа к его исходному заголовку.

По умолчанию: Вкл

Добавление автоматических субтитров

Parabolic предоставит на загрузку автоматически созданные субтитры в дополнение к доступным.

По умолчанию: Вкл

Предпочтительный видеокодек

Видеокодек, который должен быть в приоритете у Parabolic при обработке доступных к загрузке медиаформатов.

Если Parabolic не сможет найти ни одного формата с конкретно предпочтительным кодеком, ни один видеоформат не будет доступен для загрузки.

По умолчанию: Любой

Предпочтительный аудиокодек

Аудиокодек, который должен быть в приоритете у Parabolic при обработке доступных к загрузке медиаформатов.

Если Parabolic не сможет найти ни одного формата с конкретно предпочтительным кодеком, ни один аудиоформат не будет доступен для загрузки.

По умолчанию: Любой

Предпочтительный формат субтитров

Формат файла, в котором Parabolic должен будет загружать все субтитры.

По умолчанию: Любой
