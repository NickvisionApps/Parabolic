Пользовательский интерфейс

На данной странице описаны все параметры по конфигурации пользовательского интерфейса в настройках Parabolic.

Тема

Непосредственно оформление для пользовательского интерфейса.

По умолчанию: Системная

Запрет выключения

Parabolic не позволит компьютеру перейти в спящий режим во время загрузки.

По умолчанию: Выкл

Длительность истории загрузок

Срок хранения последних загрузок в истории приложения.

По умолчанию: Одна неделя
