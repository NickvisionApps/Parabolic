Новая загрузка

Эта страница поясняет все опции по настройке новой загрузки в Parabolic.

Ссылка

Ссылка для скачивания медиа. Должна быть корректной ("корректной" в смысле синтаксиса) http или https ссылкой.

Если корректная ссылка обнаружена в буфере обмена, она будет автоматически вставлена в поле при открытии диалога добавления новой загрузки.

Аутентификация

Если включено, пользователь сможет предоставить имя пользователя и/или пароль для передачи yt-dlp при загрузке.

Если включена связка ключей, пользователь сможет выбрать учётные данные из списка, либо ввести имя пользователя и/или пароль вручную.

Не все сайты поддерживают аутентификацию по имени пользователя и паролю. Например, YouTube НЕ поддерживает эту функцию.

Для сайтов не поддерживающих ввод учётных данных, рекомендуется загрузить файл куки в Настройках.

Тип файла

Файл какого типа требуется загрузить.

Если конвертирование отключено в настройках, можно будет выбрать только Видео (если доступно) и Аудио

Использованный тип файла будет сохранён и автоматически выбран в следующий раз.

Качество

Если выбранный тип файла является видео, будут перечислены доступные разрешения видео.

Если выбранный тип файла является аудио, для выбора будут доступны Лучшее и Худшее качество.

Выбранное качество будет сохранено и автоматически выбрано в следующий раз.

Загрузить субтитры

Должны ли бы загружены и встроены субтитры при скачивании видео. Языки субтитров могут быть указаны в настройках.

Выбранное состояние переключателя будет сохранено и автоматически выбрано в следующий раз.

Папка сохранения

Папка, в которую сохранить загруженный файл.

Если выбранный тип файла является аудио, для выбора будут доступны Лучшее и Худшее качество.

Выбранная папка будет сохранена и автоматически выбрана в следующий раз.

Ограничение скорости

Если включено, Parabolic ограничит скорость загрузки в соответствии с выбранным в Настройках ограничением.

Доступно в Дополнительных опциях.

Загружать в AV1

Доступно только при скачивании видео (в mp4, webm)

Если включено, Parabolic будет скачивать видео в кодеке av1, когда он доступен.

Доступно в Дополнительных опциях.

Выбор будет сохранён и поставлен автоматически в следующий раз.

Разделить по главам

Если включено, Parabolic разделит видео на части в соответствии с его оглавлением.

Доступно в Дополнительных опциях.

Обрезать миниатюру

Доступно только если Встраивание метаданных включено в Настройках.

Если включено, Parabolic обрежет миниатюру, сделав её квадратной. Полезно при скачивании музыки, миниатюры для которой обычно квадратные.

Доступно в Дополнительных опциях.

Скачать фрагмент

Если включено, можно будет выбрать время начала и конца для скачивания.

Доступно в Дополнительных опциях.

При включении этой опции НЕ будет использоваться загрузчик aria2, даже если он включён в Настройках.

Имя файла

Имя, под которым сохранить загруженный файл.
