Новая загрузка

На данной странице описаны все параметры по конфигурации новой загрузки в настройках Parabolic.

Ссылка на медиа

Непосредственно ссылка на загружаемое медиа. Должна быть корректной (в плане синтаксиса), с префиксом http или https.

Если в буфере обмена хранится корректная ссылка, она автоматически будет вставлена в поле ввода адреса при открытии диалога добавления загрузки.

Использовать пакетный файл

Выберите пакетный текстовый файл, содержащий список ссылок для проверки.

Each URL can be accompanied by an optional save folder and filename (delimited by |) in which to save each media of the URL to.

Parabolic сгруппирует и обработает все ссылки со связанными медиа из пакетного файла в единый плейлист.

Дополнительная папка сохранения должна иметь абсолютный путь.

Формат пакетного файла:

Ссылка1

URL2 | "Folder1" | Filename1

"Ссылка3"|"Папка2"

"URL4" | Folder3 | "Filename2"

Авторизация

Если параметр включён, пользователь сможет предоставить имя пользователя и/или пароль для передачи yt-dlp при загрузке.

Если учётные данные найдены в связке ключей, пользователь может выбрать учётные данные из списка (или продолжит вводить их вручную)

Не все сайты поддерживают передачу имени пользователя и пароля в yt-dlp. Например, YouTube НЕ поддерживает данную функцию.

Для сайтов, не поддерживающих передачу учётных данных, в настройках рекомендуется выбрать браузер для считывания cookie.

Быстрая загрузка

Parabolic будет загружать медиа сразу после проверки, с учётом ранее настроенных параметров загруки.

По умолчанию: Выкл

Тип файла

Файл какого типа требуется загрузить из медиа.

Ранее выбранный тип файла будет запоминаться и предварительно указываться для следующей загрузки.

Выбор типов файлов Видео (общее) или Аудио (общее) приведёт к тому, что Parabolic загрузит медиа в исходном формате без выполнения каких-либо конвертаций.

Распространённые типы файлов не полностью поддерживают встраивание обложек и субтитров. Раздельные файлы можно записать на носитель при неподдерживаемом встраивании.

Видеоформат

Список доступных для загрузки видеоформатов, отсортированных по разрешению и битрейту.

Каждый формат имеет вид: разрешение | битрейт | кодек (id).

Если выбранный тип файла является аудио, данный параметр будет недоступен.

Ранее выбранный видеоформат будет запоминаться и предварительно указываться для следующей загрузки.

Parabolic использует следующие спецформаты для соответствия поведению yt-dlp:

— Нет = Параметр не передаётся yt-dlp

— Лучший = bv*

— Худший = wv*

Аудиоформат

Список доступных для загрузки аудиоформатов, отсортированных по битрейту.

Каждый формат имеет вид: битрейт | язык (id).

Ранее выбранный аудиоформат будет запоминаться и предварительно указываться для следующей загрузки.

Parabolic использует следующие спецформаты для соответствия поведению yt-dlp:

— Нет = Параметр не передаётся yt-dlp

— Лучший = ba

— Худший = wa

Субтитры

Список доступных субтитров, которые пользователь выбирает для загрузки. Дополнительные параметры субтитров задаются в настройках.

Ранее выбранные субтитры будут запоминаться и предварительно указываться для следующей загрузки.

Разделение видео по главам

Если параметр включён, Parabolic разобьёт загружаемое видео на меньшие файлы в соответствии с его главами.

Ранее выбранное состояние будет запоминаться и предварительно указываться для следующей загрузки.

Экспорт описания

Если параметр включён, Parabolic экспортирует описание медиа в файл .description рядом с загруженным файлом.

Ранее выбранное состояние будет запоминаться и предварительно указываться для следующей загрузки.

Исключение из истории

Если параметр включён, Parabolic исключит данную загрузку из общей истории.

Аргумент постпроцессора

Аргумент, передаваемый постпроцессору при загрузке. Доступные аргументы постобработки задаются в настройках.

Ранее выбранный аргумент будет запоминаться и предварительно указываться для следующей загрузки.

Запись файла плейлиста M3U

Если параметр включён, Parabolic создаст файл m3u для загрузки плейлиста.

Ранее выбранное состояние будет запоминаться и предварительно указываться для следующей загрузки плейлиста.

Временной отрезок

Пользователь может задать промежутки начала и конца, чтобы загрузить определённый отрезок медиа.

По умолчанию: Медиа загружается целиком

Папка сохранения

Папка, куда сохранятся загружаемые файлы.

Ранее выбранная папка будет запоминаться и предварительно указываться для следующей загрузки.

Имя файла

Имя, используемое при сохранении загружаемого файла.

По умолчанию: Заголовок медиа
