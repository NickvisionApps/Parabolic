ダウンロード

このページでは Parabolic でダウンロードを設定するためのすべてオプションについて説明します。

アクティブダウンロードの最大数

平行して進行できるアクティブダウンロード数。数値は1から10の整数である必要があります。

アクティブダウンロードが最大数に達すると、Parabolic は新しいダウンロードをダウンロードが可能になるまでキューに入れます。

デフォルト: 5

存在するファイルを上書きする

有効であれば、Parabolic は存在するファイルを上書きします。

例えば、movie.mp4という名前のファイルがダウンロードフォルダーに存在し、movieという名前のダウンロードが作成されると、movie.mp4は新しいダウンロードの内容で上書きされます。

Parabolic は、再開する必要のあるダウンロードを検知し、失敗したダウンロードの再開時または再試行時にこの設定を無視します

デフォルト: オン

ファイル名の文字を制限

ファイル名の文字を Windows がサポートする文字のみに制限する。

このオプションは Linux でのみ利用できます。

デフォルト: オフ

検証完了後のタイトルにメディア ID を含める

検証後にメディアのタイトル（ファイル名）にメディアの ID を含めるかどうか。

デフォルト: オン

自動生成された字幕を含める

要求された字幕の言語の自動翻訳された字幕を含むかどうか。

デフォルト: オン

優先するビデオコーデック

ダウンロードするメディアのフォーマットの選択時に、Parabolic が使用するビデオコーデック。

デフォルト: すべて

優先する字幕のフォーマット

Parabolic がダウンロードするすべての字幕のファイル形式。

デフォルト: すべて
