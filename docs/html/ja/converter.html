コンバーター

このページでは Parabolic でコンバーターを設定するためのすべてオプションについて説明します。

メタデータを埋め込む

If enabled, Parabolic will tag a media with its relevant information (title, genre, etc...) if available.

デフォルト: オン

ダウンロード元のデータを削除

有効であれば、Parabolic はメディアのソースを特定できる URL などのメタデータを削除します。

これらのメタデータ欄: comment、description、synopsis、purl、handler_name は初めから埋め込まれません。

このオプションは メタデータを埋め込む が有効な場合のみ設定可能です。

デフォルト: オフ

字幕を埋め込む

If enabled, Parabolic will embed a media with its thumbnail if available.

ファイル形式が埋め込みに対応していない場合は、サムネイルは独立した画像ファイルとして保存されます。

デフォルト: オン

オーディオサムネイルを整形

有効であれば、Parabolic はオーディオファイルのサムネイル画像を正方形に切り抜きます。

This option is only configurable if Embed Thumbnails is on.

デフォルト: オフ

チャプター（章）を埋め込む

有効であれば、利用可能であれば、Parabolic はメディアにチャプター（章）をタグ付けします。

デフォルト: オフ

字幕を埋め込む

無効化されていたり、埋め込みが対応されていない場合は、ダウンロードされた字幕は独立したファイルに保存されます。

デフォルト: オン

ポストプロセッサのスレッド

各ダウンロードで使用するポストプロセッサ (FFmpeg) に許可するスレッドの数。

デフォルト: システムで利用できるスレッドの最大値
