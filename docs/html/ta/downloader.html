பதிவிறக்குபவர்

பரபோலிக் இல் உள்ள விருப்பங்களில் பதிவிறக்குபவரை உள்ளமைப்பதற்கான அனைத்து விருப்பங்களையும் இந்தப் பக்கம் விளக்குகிறது.

Use Part Files

Parabolic will download media in separate .part files instead of directly into the output file.

இயல்புநிலை: ஆன்

YouTubeக்கு SponsorBlock ஐப் பயன்படுத்தவும்

Parabolic will attempt to remove sponsored segments from YouTube videos.

YouTube வீடியோக்களில் மட்டுமே SponsorBlock வேலை செய்யும். இருப்பினும், இந்த அமைப்பை ஆன் செய்வதால் பிற தளங்களில் இருந்து பதிவிறக்கம் பாதிக்கப்படாது (அதாவது அவை தோல்வியடையாது).

இயல்புநிலை: ஆஃப்

வேக வரம்பு

The limit, in KiB/s, to apply to downloads only with the advanced option Speed Limit enabled. Must be an integer value between 512 and 10240.

இயல்புநிலை: 1024

ப்ராக்ஸி URL

பயன்படுத்த வேண்டிய ப்ராக்ஸி சேவையகத்தின் url.

இயல்புநிலை: முகவரி இல்லை

உலாவியில் இருந்து குக்கீகள்

குக்கீகளைப் படித்து YT-DLP இன் பதிவிறக்க இயந்திரத்திற்கு அனுப்ப ஒரு உலாவி.

உள்நுழைவு தேவைப்படும் மீடியாவைப் பதிவிறக்கும்போது இது பயனுள்ளதாக இருக்கும், எடுத்துக்காட்டாக: தனிப்பட்ட YouTube வீடியோக்கள்.

இந்த விருப்பம் சாண்ட்பாக்ச் அல்லாத தளங்களில் கிடைக்கிறது.

இயல்புநிலை: எதுவுமில்லை

குக்கீகள் கோப்பு

பதிவிறக்க இயந்திரத்தைப் பதிவிறக்கும்போது YT-DLP க்கு அனுப்ப குக்கீகளின் .txt கோப்பு.

உள்நுழைவு தேவைப்படும் மீடியாவைப் பதிவிறக்கும்போது இது பயனுள்ளதாக இருக்கும், எடுத்துக்காட்டாக: தனிப்பட்ட YouTube வீடியோக்கள்.

உங்கள் உலாவியில் இருந்து குக்கீகள் கோப்பைப் பெற, பின்வரும் நீட்டிப்புகளில் ஒன்றைப் பயன்படுத்தவும்:

குரோம்: https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc

பயர்பாக்ஸ்: https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/

இந்த நீட்டிப்புகளை உங்கள் சொந்த ஆபத்தில் பயன்படுத்தவும்.

இயல்புநிலை: கோப்பு எதுவும் தேர்ந்தெடுக்கப்படவில்லை
