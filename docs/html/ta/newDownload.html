புதிய பதிவிறக்கம்

பரபோலிக் இல் புதிய பதிவிறக்கத்தை உள்ளமைப்பதற்கான அனைத்து விருப்பங்களையும் இந்தப் பக்கம் விளக்குகிறது.

ஊடகத்தின் URL

பதிவிறக்குவதற்கான மீடியாவின் URL. செல்லுபடியாகும் ("செல்லுபடியாகும்" அதாவது தொடரியல் ரீதியாக சரியானது) http அல்லது https இணைப்பாக இருக்க வேண்டும்.

பதிவிறக்கத்தைச் சேர் என்ற உரையாடலைத் திறந்தவுடன் கிளிப்போர்டுக்குள் சரியான URL கண்டறியப்பட்டால், URL நுழைவு வரிசையில் தானாகவே URL ஒட்டப்படும்.

அங்கீகரிக்கவும்

இயக்கப்பட்டால், பதிவிறக்கும் போது yt-dlp க்கு அனுப்ப பயனர் பெயர் மற்றும்/அல்லது கடவுச்சொல்லை வழங்குமாறு பயனர் கேட்கப்படுவார்.

சாவி வளையம் இயக்கப்பட்டிருந்தால், பெயரிடப்பட்ட நற்சான்றிதழ்களின் பட்டியலிலிருந்து தேர்வு செய்யும்படி பயனர் கேட்கப்படுவார் அல்லது வேறு பயனர்பெயர் மற்றும்/அல்லது கடவுச்சொல்லை கைமுறையாக உள்ளிட தேர்வு செய்யலாம்.

அனைத்து இணையதளங்களும் yt-dlp க்கு பயனர்பெயர்கள் மற்றும் கடவுச்சொற்களை அனுப்புவதை ஆதரிக்காது. எடுத்துக்காட்டாக, YouTube இந்த அம்சத்தை ஆதரிக்காது.

நற்சான்றிதழ்களை அனுப்புவதை ஆதரிக்காத இந்தத் தளங்களுக்கு, விருப்பத்தேர்வுகளில் குக்கீகள் கோப்பைப் பதிவேற்ற பரிந்துரைக்கப்படுகிறது.

கோப்பு வகை

மீடியாவைப் பதிவிறக்க வேண்டிய கோப்பு வகை.

Disallow Conversions இயக்கப்பட்டிருந்தால், Video (ஆதரிக்கப்பட்டால்) மற்றும் Audio மட்டுமே விருப்பங்கள் இருக்கும்

முன்பு பயன்படுத்தப்பட்ட கோப்பு வகை நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

தரம்

கோப்பு வகை வீடியோ வகையாக இருந்தால், பட்டியலிடப்பட்ட குணங்கள் மீடியாவின் தீர்மானங்களாக இருக்கும்.

கோப்பு வகை ஆடியோ வகையாக இருந்தால், பட்டியலிடப்பட்ட குணங்கள் வெறுமனே Best மற்றும் Worst இருக்கும்.

முன்பு பயன்படுத்தப்பட்ட தரம் நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

Download Subtitle

Whether to download and embed subtitle when downloading a video. Languages for subtitles can be defined in preferences.

The previously used state will be remembered and pre-selected for the next download.

கோப்புறையை சேமி

பதிவிறக்கத்தைச் சேமிக்க வேண்டிய கோப்புறை.

கோப்பு வகை ஆடியோ வகையாக இருந்தால், பட்டியலிடப்பட்ட குணங்கள் வெறுமனே Best மற்றும் Worst இருக்கும்.

முன்பு பயன்படுத்திய கோப்புறை நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

வேக வரம்பு

இயக்கப்பட்டால், பரபோலிக் பதிவிறக்க வேகத்தை விருப்பங்களில் உள்ளமைக்கப்பட்ட வேகத்திற்கு வரம்பிடும்.

Advanced Options பிரிவில் கிடைக்கும்.

Prefer AV1 Codec

Only available for video (mp4, webm) downloads

If enabled, Parabolic will prefer videos with the av1 codec when downloading.

Advanced Options பிரிவில் கிடைக்கும்.

This option's state will be remembered and restored for future downloads.

அத்தியாயங்களைப் பிரிக்கவும்

இயக்கப்பட்டால், பரபோலிக் வீடியோவை அதன் அத்தியாயங்களின் அடிப்படையில் பல சிறியதாகப் பிரிக்கும்.

Advanced Options பிரிவில் கிடைக்கும்.

சிறுபடத்தை வெட்டவும்

Embed Metadata விருப்பத்தேர்வுகளில் இயக்கப்பட்டிருந்தால் மட்டுமே கிடைக்கும்.

இயக்கப்பட்டால், பரபோலிக் பதிவிறக்கத்தின் உட்பொதிக்கப்பட்ட சிறுபடத்தை சதுரமாகச் செதுக்கும். ஆல்பம் படங்கள் பொதுவாக சதுரமாக இருக்கும் இசையைப் பதிவிறக்கும் போது இது பயனுள்ளதாக இருக்கும்.

குறிப்பிட்ட காலக்கெடுவைப் பதிவிறக்கவும்
