புதிய பதிவிறக்கம்

பரபோலிக் இல் புதிய பதிவிறக்கத்தை உள்ளமைப்பதற்கான அனைத்து விருப்பங்களையும் இந்தப் பக்கம் விளக்குகிறது.

ஊடகத்தின் URL

பதிவிறக்குவதற்கான மீடியாவின் URL. செல்லுபடியாகும் ("செல்லுபடியாகும்" அதாவது தொடரியல் ரீதியாக சரியானது) http அல்லது https இணைப்பாக இருக்க வேண்டும்.

பதிவிறக்கத்தைச் சேர் என்ற உரையாடலைத் திறந்தவுடன் கிளிப்போர்டுக்குள் சரியான URL கண்டறியப்பட்டால், URL நுழைவு வரிசையில் தானாகவே URL ஒட்டப்படும்.

அங்கீகரிக்கவும்

இயக்கப்பட்டால், பதிவிறக்கும் போது yt-dlp க்கு அனுப்ப பயனர் பெயர் மற்றும்/அல்லது கடவுச்சொல்லை வழங்குமாறு பயனர் கேட்கப்படுவார்.

If credentials are found in the keyring, the user can instead chose from a list of named credentials to use (or continue to manually enter a credential)

அனைத்து இணையதளங்களும் yt-dlp க்கு பயனர்பெயர்கள் மற்றும் கடவுச்சொற்களை அனுப்புவதை ஆதரிக்காது. எடுத்துக்காட்டாக, YouTube இந்த அம்சத்தை ஆதரிக்காது.

For these sites that don't support passing credentials, it is recommended to select a browser to use cookies from in Preferences instead.

கோப்பு வகை

மீடியாவைப் பதிவிறக்க வேண்டிய கோப்பு வகை.

Disallow Conversions இயக்கப்பட்டிருந்தால், Video (ஆதரிக்கப்பட்டால்) மற்றும் Audio மட்டுமே விருப்பங்கள் இருக்கும்

முன்பு பயன்படுத்தப்பட்ட கோப்பு வகை நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

தரம்

கோப்பு வகை வீடியோ வகையாக இருந்தால், பட்டியலிடப்பட்ட குணங்கள் மீடியாவின் தீர்மானங்களாக இருக்கும்.

If the file type is an audio type, the listed qualities will be simply Best, Good, and Worst.

முன்பு பயன்படுத்தப்பட்ட தரம் நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

Audio Language

The language of which to use from the audio part of a media download.

If no additional languages are available, Default will be the only option.

The previously used state will be remembered and pre-selected for the next download.

Download Subtitle

Whether to download and embed subtitle when downloading a video. Further subtitle settings can be configured in preferences.

The previously used state will be remembered and pre-selected for the next download.

Video Codec

The video codec to prefer when downloading a video.

The previously used codec will be remembered and pre-selected for the next download.

அத்தியாயங்களைப் பிரிக்கவும்

If enabled, Parabolic will split a video download into multiple smaller files based on its chapters.

The previously used state will be remembered and pre-selected for the next download.

கோப்புறையை சேமி

பதிவிறக்கத்தைச் சேமிக்க வேண்டிய கோப்புறை.

முன்பு பயன்படுத்திய கோப்புறை நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

கோப்பு பெயர்

பதிவிறக்கத்தைச் சேமிக்கும்போது பயன்படுத்த வேண்டிய கோப்பு பெயர்.

இயல்புநிலை: ஊடகத்தின் தலைப்பு

Time Frame

The user can enter a start time frame and an end time frame to download a specific section of the media.

Default: The whole media length

Enabling this option will disable the use of aria2 as the downloader for this download, even if it is enabled in Preferences.

வேக வரம்பு

If enabled, Parabolic will limit the download's speed with the speed configured in Preferences.
