புதிய பதிவிறக்கம்

பரபோலிக் இல் புதிய பதிவிறக்கத்தை உள்ளமைப்பதற்கான அனைத்து விருப்பங்களையும் இந்தப் பக்கம் விளக்குகிறது.

ஊடகத்தின் URL

பதிவிறக்குவதற்கான மீடியாவின் URL. செல்லுபடியாகும் ("செல்லுபடியாகும்" அதாவது தொடரியல் ரீதியாக சரியானது) http அல்லது https இணைப்பாக இருக்க வேண்டும்.

பதிவிறக்கத்தைச் சேர் என்ற உரையாடலைத் திறந்தவுடன் கிளிப்போர்டுக்குள் சரியான URL கண்டறியப்பட்டால், URL நுழைவு வரிசையில் தானாகவே URL ஒட்டப்படும்.

Use Batch File

Select a batch text file that contains a list of URLs to validate.

Each URL can be accompanied by an optional save folder (delimited by |) in which to save each media of the URL to.

All of the URLs and their media from the batch file will be grouped and treated as one single playlist by Parabolic.

The optional save folder must be an absolute path.

Batch File Format:

URL1

URL2 | "Folder1"

"URL3"|"Folder2"

"URL4"

அங்கீகரிக்கவும்

இயக்கப்பட்டால், பதிவிறக்கும் போது yt-dlp க்கு அனுப்ப பயனர் பெயர் மற்றும்/அல்லது கடவுச்சொல்லை வழங்குமாறு பயனர் கேட்கப்படுவார்.

கெய்ரிங்கில் நற்சான்றிதழ்கள் காணப்பட்டால், பயனர் பயன்படுத்த பெயரிடப்பட்ட நற்சான்றிதழ்களின் பட்டியலிலிருந்து தேர்வு செய்யலாம் (அல்லது தொடர்ந்து நற்சான்றிதழை கைமுறையாக உள்ளிடவும்)

அனைத்து இணையதளங்களும் yt-dlp க்கு பயனர்பெயர்கள் மற்றும் கடவுச்சொற்களை அனுப்புவதை ஆதரிக்காது. எடுத்துக்காட்டாக, YouTube இந்த அம்சத்தை ஆதரிக்காது.

கடந்து செல்லும் நற்சான்றிதழ்களை ஆதரிக்காத இந்த தளங்களுக்கு, அதற்கு பதிலாக குக்கீகளை விருப்பங்களிலிருந்து பயன்படுத்த உலாவியைத் தேர்ந்தெடுக்க பரிந்துரைக்கப்படுகிறது.

Download Immediately

Parabolic will immediately download the media after validation, using the previously configured download options.

இயல்புநிலை: ஆஃப்

கோப்பு வகை

மீடியாவைப் பதிவிறக்க வேண்டிய கோப்பு வகை.

முன்பு பயன்படுத்தப்பட்ட கோப்பு வகை நினைவில் வைக்கப்படும் மற்றும் அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

வீடியோ (பொதுவான) அல்லது ஆடியோ (பொதுவான) கோப்பு வகைகளைத் தேர்ந்தெடுப்பது எந்தவொரு மாற்றங்களையும் செய்யாமல் மீடியாவின் அசல் கோப்பு வகைகளில் ஊடகத்தை பதிவிறக்கம் செய்ய பரபோலிக் ஏற்படுத்தும்.

Generic file types do not fully support embedding thumbnails and subtitles. Separate files may be written to disk if embedding is not supported.

வீடியோ வடிவம்

பதிவிறக்கத்திற்கு கிடைக்கும் வீடியோ வடிவங்களின் பட்டியல், தீர்மானம் மற்றும் பிட்ரேட் ஆகியவற்றால் வரிசைப்படுத்தப்பட்டது.

ஒவ்வொரு வடிவமும் படிவம்: தீர்மானம் | பிட்ரேட் | கோடெக் (ஐடி) .

கோப்பு வகை ஆடியோ வகை என்றால், இந்த விருப்பம் கிடைக்காது.

முன்னர் பயன்படுத்தப்பட்ட வீடியோ வடிவம் நினைவில் வைத்து அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

The following are special formats used by Parabolic to match yt-dlp behaviors:

- None = No option passed to yt-dlp

- Best = bv*

- Worst = wv*

ஆடியோ வடிவம்

பிட்ரேட்டால் வரிசைப்படுத்தப்பட்ட பதிவிறக்கத்திற்கு கிடைக்கும் ஆடியோ வடிவங்களின் பட்டியல்.

ஒவ்வொரு வடிவமும் படிவம்: பிட்ரேட் | மொழி (ஐடி) .

முன்னர் பயன்படுத்தப்பட்ட ஆடியோ வடிவம் நினைவில் வைத்து அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

The following are special formats used by Parabolic to match yt-dlp behaviors:

- None = No option passed to yt-dlp

- Best = ba

- Worst = wa

வசன வரிகள்

பயனரால் பதிவிறக்கம் செய்ய தேர்ந்தெடுக்கக்கூடிய ஊடகங்களால் கிடைக்கும் வசன வரிகள் பட்டியல். மேலும் வசன அமைப்புகளை விருப்பத்தேர்வுகள் இல் கட்டமைக்க முடியும்.

முன்னர் தேர்ந்தெடுக்கப்பட்ட வசன வரிகள் நினைவில் வைக்கப்பட்டு அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

அத்தியாயங்களைப் பிரிக்கவும்

இயக்கப்பட்டால், பரபோலிக் ஒரு வீடியோ பதிவிறக்கத்தை அதன் அத்தியாயங்களின் அடிப்படையில் பல சிறிய கோப்புகளாக பிரிக்கும்.

முன்னர் பயன்படுத்தப்பட்ட நிலை நினைவில் வைக்கப்பட்டு அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

ஏற்றுமதி விளக்கம்

இயக்கப்பட்டால், பரபோலிக் ஊடகத்தின் விளக்கத்தை க்கு ஏற்றுமதி செய்யும். விளக்கப்படம் கோப்புடன் பதிவிறக்கம் செய்யப்பட்ட கோப்புடன்.

முன்னர் பயன்படுத்தப்பட்ட நிலை நினைவில் வைக்கப்பட்டு அடுத்த பதிவிறக்கத்திற்கு முன்பே தேர்ந்தெடுக்கப்படும்.

Exclude from History

If enabled, Parabolic will exclude the download from the list of historic downloads.

Post Processor Argument

An argument to pass to the post processor during the download. The available postprocessing arguments can be configured in preferences.

The previously selected argument will be remembered and pre-selected for the next download.

Write M3U Playlist File

If enabled, Parabolic will write a m3u file for the playlist download.

The previously used state will be remembered and pre-selected for the next playlist download.

கால அளவு

மீடியாவின் ஒரு குறிப்பிட்ட பகுதியைப் பதிவிறக்க பயனர் தொடக்க கால காலநீடிப்பு மற்றும் இறுதி கால கட்டத்தை உள்ளிடலாம்.
