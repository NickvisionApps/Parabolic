Завантажувач

На цій сторінці описано всі параметри налаштування завантажувача у параметрах Parabolic.

Перезаписати наявні файли

Якщо ввімкнено, Parabolic перезапише наявні файли.

Наприклад, якщо в теці для завантаження існує файл з іменем movie.mp4, а в теці створюється нове завантаження з іменем movie, movie.mp4 буде замінено вмістом нового завантаження.

За замовчуванням: Ввімк

Максимальна кількість активних завантажень

Кількість активних завантажень, які дозволено виконувати одночасно. Має бути цілим числом від 1 до 10.

Якщо досягнуто максимальної кількості активних завантажень, Parabolic ставитиме нові завантаження в чергу, доки не з'явиться більше завантажень, які можна буде запустити.

За замовчуванням: 5

Мови субтитрів

Список розділених комами з двох літер, двох літер з регіоном та трилітерних кодів мови, які слід використовувати при виборі мов субтитрів для завантаження. Застосовується тільки при завантаженні відео з субтитрами.

Різні сайти можуть використовувати різні формати коду мови для однієї мови. Ви можете зіткнутися з одним з наступних трьох типів: дволітерний, дволітерний з регіоном або трилітерний. Наприклад, «en», «en-US» і «eng» всі використовуються для англійської мови. Будь ласка, вкажіть всі дійсні коди для ваших мов для найкращих результатів. Якщо є, автоматично створені субтитри також будуть завантажені.

Ви можете вказати «всі» замість завантаження всіх субтитрів, але без автоматично згенерованих.

За замовчуванням: коди мови вашої культури. Наприклад, в системі en-US за замовчуванням буде en,en-US,eng

Вмикати автоматично згенеровані субтитри

Чи треба вмикати субтитри, створені YouTube автоматично, для покриття обраних мов субтитрів.

За замовчуванням: Ввімк

Використовувати aria2

Якщо ввімкнено, Parabolic використовуватиме aria2c для завантаження медіафайлів. Хоча використання aria2c може покращити швидкість завантаження, хід завантаження не відображатиметься у користувацькому інтерфейсі.

За замовчуванням: Вимк

Максимальна кількість з'єднань на сервер

Цей параметр застосовується лише тоді, коли ввімкнено використання aria2. Цей параметр є аналогом прапорця -x, переданого до aria2c.

Максимальна кількість з'єднань, яку можна використовувати для одного сервера. Має бути цілим числом від 1 до 32.

За замовчуванням: 16

Мінімальний розмір розділення

Цей параметр застосовується лише тоді, коли ввімкнено використання aria2. Цей параметр є аналогом прапорця -k, переданого до aria2c.

Мінімальний розмір, у МБ, на який можна розбити файл. Має бути цілим числом від 1 до 1024.

За замовчуванням: 20

Обмеження швидкості

Обмеження в кб/с, яке застосовується до завантажень, тільки якщо для них ввімкнено додаткову опцію Обмеження швидкості. Повинно бути цілим числом від 512 до 10240.

За замовчуванням: 1024

Проксі URL

URL-адреса проксі-сервера для використання.

Файл cookie

Файл .txt з файлами cookie для передачі yt-dlp під час завантаження. Це може бути корисно при завантаженні медіа, які вимагають входу, наприклад: приватні відео на YouTube.

Використовуйте одне з наступних розширень для отримання файлів cookie з вашого браузера:

Chrome: https://chrome.google.com/webstore/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc

Firefox: https://addons.mozilla.org/en-US/firefox/addon/cookies-txt/

Використовуйте ці розширення на свій страх і ризик.

За замовчуванням: Файл не вибрано

Використовувати SponsorBlock для YouTube

Якщо ввімкнено, Parabolic застосує розширення SponsorBlock до завантажень YouTube.

SponsorBlock працює лише на відео YouTube. Однак залишення цього налаштування ввімкненим не вплине на завантаження з інших сайтів (тобто не призведе до їх невдачі).

За замовчуванням: Вимк
