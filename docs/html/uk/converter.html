Конвертер

Ця сторінка пояснює всі опції для налаштування конвертера в Параметрах Parabolic.

Вставити метадані

Якщо увімкнено, Parabolic додасть до медіа відповідні теги (назва, жанр тощо), якщо вони доступні.

За замовчуванням: Ввімк

Вилучити вихідні дані

Якщо ввімкнено, Parabolic очистить поля метаданих, що містять URL та іншу ідентифікаційну інформацію джерела медіа.

Очищені поля: коментар, опис, синопсис, purl і handler_name в кожному стримі.

Цей параметр можна налаштувати лише, якщо увімкнено Вбудовувати метадані.

За замовчуванням: Вимк

Вбудовані ескізи

Якщо увімкнено, Parabolic вбудує ескіз у медіа, якщо він доступний.

Якщо тип файлу не підтримує вбудовування, ескіз буде збережено окремим зображенням.

За замовчуванням: Ввімк

Обрізати мініатюри аудіо

Якщо ввімкнено, Parabolic обрізає ескізи для аудіофайлів до квадратної роздільної здатності.

Цю опцію можна налаштувати лише при ввімкненій функції Вбудовувати ескізи.

За замовчуванням: Вимк

Вставити розділи

Якщо увімкнено, Parabolic позначить медіа інформацією про розділи, якщо така доступна.

За замовчуванням: Вимк

Вбудовані субтитри

Якщо вимкнено або вбудовування не підтримується, завантажені субтитри зберігатимуться окремим файлом.

За замовчуванням: Ввімк

Потоки постобробки

Кількість потоків, які можна дозволити постпроцесору (FFmpeg) використовувати для кожного завантаження.

За замовчуванням: максимальна кількість потоків, доступна в системі.
